Instalación

Para usar Less en producción, es recomandado usarlo con Node.js con los comandos:

npm install -g less > lessc styles.less styles.css

Para compilar lo directo en navegador, se puede colocar en el elemento <head> de la siguiente forma:

<link rel="stylesheet/less" type="text/css" href="styles.less"/> <script src="https://cdn.jsdelivr.net/npm/less@4"></script>

Si estas intentando de usar less sin un servidor, y ves una pagina blanca, puedes poner -allow-file-access-from-files en el shortcut de tu navegador.

Features

Variables

Less te permite declarar variables que se pueden usar como valores, selectores, propiedades, URLs, y @import

@e declararon con el uso del simbolo @ antes del nombre del variable, por ejemplo: el color de fondo esta declarado con @bg: #121517;

Scope: si reasignas un variable dentro de un ruleset, llamando el variable busca primera dentro del ruleset y despues los rulesets padres (nesting)

Mixins

Los Mixins te permiten usar las propiedades de un ruleset dentro de otro.

Para usar los Mixins, simplemente pones el selector con parentesis como si fuera una propiedad sin valor:

.highlight {
    color: red;
    font-weight: bold;
}
p span {
    text-decoration: underline;
    .highlight();
}

Tambien puedes agrupar tus mixins para organizacion

Si agrupas .boton y .radio dentro de #inputs, puedes usar los de forma: #inputs.boton(); y #inputs.radio();

Nesting

Less te permite anidar rulesets dentro de otros rulesets

Incluso funciona con @media, @support selectores, y pseudo selectores:

ul li{
    display: inline;
    a {
        color: red;
        text-decoration: none;
        &:hover {
            color: green;
        }
    }
    @media (min-width: 768px) {
        display: block;
    }
}

Operaciones

Less te permite realizar operaciones entre numeros de diferente unidades y los convierte al tipo del primer numero, si conversion no es posible ignora los tipos.

Tambien te permite realizar operaciones con colores.

background-color: (#223344 * 2); resulta en #446688

Funciones

Less te permite usar funciones logicas, funciones de String, funciones matematicas, funciones de color, y varios otros.

Con estas funciones puedes hacer condicionales, modificar colores, modificar texto, generar listas con rangos, modificar numeros, y mucho mas
